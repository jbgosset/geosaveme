# Functional Limits

## Technical Limits

| Constraint | Description |
|---|---|
| **Train, metro, tram** | Geolocation issues caused by rapid movement through urban infrastructure. |
| **Multi-storey buildings** | Geolocation issues related to elevation — GPS cannot reliably distinguish between floors. |

## Social / Contextual Limits

| Constraint | Description |
|---|---|
| **Vehicle (car)** | A private, enclosed space in motion — alert emission is physically constrained and geolocation may be imprecise at speed. |
