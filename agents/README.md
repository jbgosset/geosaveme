# Agents

Instructions, scripts, and sub-agent definitions for all AI agents contributing to this repository.

## Contents

| File / Folder | Purpose |
|---------------|----------|
| `AGENT.md` | **Start here** — universal rules for all AI agents and human contributors |
| `scripts/` | Utility scripts: environment setup (`setup-mac.sh`), AI commit helper (`ai-commit.sh`) |
| `sub-agents/` | Specialised sub-agent definitions (BA, security, architecture, QA…) |

## Rule

Any AI agent working on this codebase must read `AGENT.md` in full before performing any task.  
Sub-agents must also comply with all rules in `AGENT.md` unless explicitly overridden in their own definition file.
